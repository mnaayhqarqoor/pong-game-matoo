# تطبيق تتبع الوجه

هذا هو تطبيق ويب بسيط لتتبع الوجه باستخدام كاميرا الجهاز ومكتبة face-api.js.

## الميزات

- تتبع الوجه في الوقت الفعلي
- رسم مربعات حول الوجوه المكتشفة
- رسم ملامح الوجه
- واجهة باللغة العربية

## كيفية الاستخدام

1. قم بتشغيل الخادم المحلي أو رفع الملفات إلى خدمة مثل GitHub Pages
2. افتح المتصفح وانتقل إلى صفحة التطبيق
3. انقر على زر "بدء الكشف عن الوجه"
4. اسمح بالوصول إلى الكاميرا عند المطالبة
5. شاهد تتبع الوجه في الوقت الفعلي

## الملفات

- `index.html`: الصفحة الرئيسية للتطبيق
- `models/`: مجلد يحتوي على نماذج التعرف على الوجه (يتم تحميلها من CDN في هذا الإصدار)

## المكتبات المستخدمة

- [face-api.js](https://github.com/justadudewhohacks/face-api.js): مكتبة JavaScript للكشف عن الوجه والتعرف عليه

## المتصفحات المدعومة

- Chrome
- Firefox
- Edge
- Safari (قد تكون هناك بعض القيود)

## ملاحظات

- يتطلب اتصالاً بالإنترنت لتحميل نماذج التعرف على الوجه
- يجب استخدام HTTPS عند نشر التطبيق (مهم لمتصفح Chrome)
